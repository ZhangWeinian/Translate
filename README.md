# 做出最优秀的翻译软件！

* 这是一个个人项目，了学习和练习而创建的，目的是为了做出最优秀的翻译软件 \^o\^

目前是这样的计划：
1. 整个软件分为如下几个部分：
	- 对外调用接口：包含命令行调用；用 Qt6 做的 GUI 调用；用 WinUI3 做的更现代化的 GUI 调用
	- 翻译引擎：调用百度翻译 API
	- 数据储存：包含用户信息；数据处理
	- 翻译方式：包含文本翻译；图片翻译；语音翻译
	- 其他：包含日志记录；错误处理；多语言支持；多平台支持

2. 项目的目标是做出一个翻译软件，可以在 Windows、Linux、MacOS 上运行，支持多种语言，支持多种翻译方式，支持多种翻译引擎，支持多种数据储存方式，支持多种调用方式。
